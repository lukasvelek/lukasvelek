SkyDocu

SkyDocu is a containerized Document Management System.

It is like a standard Document Management System (further as DMS), but it runs under a "superadministration" that allows creation and management of several of these DMS or containers.

It is based on the newest version of my framework.

DMS (Document Management System)

×

Document Management System, or DMS, is used for managing documents.

It allows creating folders and thus prefilter documents. Administrator can also create custom metadata with predefined types and allow their usage in folders.

It is based on my own framework that I created for this projects from scratch and that allows utilization of jQuery and AJAX.

More about the project can be found here.
